# Python命令行FTP工具设计计划

## 1. 项目概述

创建一个基于Python的命令行工具，使用ftplib库实现与ZOS FTP服务器之间的文件上传和下载功能。

## 2. 仓库研究结论

当前项目目录 `e:\learn_python\mytools` 中存在一个空的 `ftptool` 目录，需要在此基础上构建完整的FTP工具。

## 3. 项目结构设计

```
ftptool/
├── ftpclient.py        # 核心FTP客户端类
├── cli.py              # 命令行接口
├── utils.py            # 工具函数
├── config.py           # 配置管理
├── requirements.txt    # 依赖管理
└── README.md           # 项目说明
```

## 4. 功能设计

### 4.1 核心功能
- 连接到ZOS FTP服务器
- 上传文件到服务器
- 从服务器下载文件
- 列出服务器目录内容
- 支持基本的FTP命令

### 4.2 命令行参数
- `--host`：FTP服务器地址
- `--port`：FTP服务器端口（默认21）
- `--user`：用户名
- `--password`：密码
- `--action`：操作类型（upload/download/list）
- `--local`：本地文件/目录路径
- `--remote`：远程文件/目录路径
- `--passive`：使用被动模式（默认True）
